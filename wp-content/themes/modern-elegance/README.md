# Modern Elegance WordPress Theme

Modern Elegance ist ein zeitgemäßes Block-Theme für WordPress, das klare Typografie, großzügige Abstände und eine ausdrucksstarke Farbpalette kombiniert. Es wurde speziell für kreative Agenturen, Start-ups und Beratungen entwickelt, die ein visuell prägnantes, aber dennoch leicht anpassbares Frontend benötigen.

## Features

- Vollständige Unterstützung für den Block-Editor inklusive eigener Block-Voreinstellungen.
- Anpassbare Farbpalette, Typografie und Abstände via `theme.json`.
- Startseiten-Template mit Hero-Bereich, Feature-Kacheln, Projektübersicht und Call-to-Action.
- Eigene Template-Parts für Header und Footer inklusive Social-Links und Kontaktbereich.
- Responsives Grid-Layout mit sanften Animationen (deaktiviert bei „prefers-reduced-motion“).
- Vorbereitung für Google Fonts (Inter & Playfair Display) und benutzerdefinierte Logos.

## Installation

1. Kopieren Sie den Ordner `modern-elegance` in das Verzeichnis `wp-content/themes/` Ihrer WordPress-Installation.
2. Melden Sie sich im WordPress-Dashboard an und aktivieren Sie das Theme unter **Design → Themes**.
3. Importieren Sie optional Demo-Inhalte oder passen Sie die Templates mit dem Site-Editor an.

## Entwicklungsnotizen

- Globale Styles und Animationen befinden sich in `assets/css/global.css`.
- JavaScript-Interaktionen (Intersection Observer) liegen in `assets/js/main.js`.
- Die wichtigsten Templates finden Sie im Ordner `templates/`, zusätzliche Template-Parts in `parts/`.
- Das Theme setzt WordPress >= 6.2 und PHP >= 7.4 voraus.
