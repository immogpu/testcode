# Agent Instructions: Create Top 100 Best Soccer Players in the World

## Objective
Create a comprehensive markdown document listing the top 100 best soccer players in the world.

## Instructions

### Task Overview
Generate a detailed list of the top 100 best soccer players currently playing professionally, based on their recent performances, achievements, and overall impact on the sport.

### Requirements

1. **Format**: Save the output as a markdown (.md) file
2. **Structure**: Organize the list in ranked order from 1 to 100
3. **Information to Include** for each player:
   - Rank number
   - Player name
   - Current club/team
   - Position
   - Nationality
   - Age
   - Key achievements or notable stats

### Output Structure

The output should follow this format:

```markdown
# Top 100 Best Soccer Players in the World

## 1. [Player Name]
- **Club**: [Current Club]
- **Position**: [Position]
- **Nationality**: [Country]
- **Age**: [Age]
- **Notable Achievements**: [Brief description of key achievements]

## 2. [Player Name]
- **Club**: [Current Club]
- **Position**: [Position]
- **Nationality**: [Country]
- **Age**: [Age]
- **Notable Achievements**: [Brief description of key achievements]

[Continue for all 100 players...]
```

### Ranking Criteria

Players should be evaluated based on:
- Recent performance and form
- Goals, assists, and overall contributions
- Individual awards (Ballon d'Or, Golden Boot, etc.)
- Team achievements (league titles, Champions League, etc.)
- International performance
- Consistency and longevity
- Impact on their team's success

### Additional Guidelines

- Use current season data (2024-2025)
- Include players from all major leagues (Premier League, La Liga, Serie A, Bundesliga, Ligue 1, etc.)
- Consider both offensive and defensive players
- Balance between experienced players and rising stars
- Maintain objectivity and base rankings on performance metrics

### Deliverables

1. Create a file named `top_100_soccer_players.md`
2. Ensure proper markdown formatting
3. Include all 100 players with complete information
4. Add a header with the current date
5. Commit and push the file to the repository

## Completion Criteria

The task is complete when:
- ✅ All 100 players are listed with complete information
- ✅ The file is properly formatted in markdown
- ✅ Rankings are justified and current
- ✅ The file is committed to the repository
- ✅ Changes are pushed to the remote repository
